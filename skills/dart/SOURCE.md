**DART** (Deconstruct / Analyze / Recognize / Test) and the four-systems framing are taken from the talk *"How To Think SO CLEARLY People Assume You're A Genius"* (YouTube `mjTgkm-h__M`). DART is the speaker's own coined mnemonic; the four "systems" (Clear / Complicated / Complex / Chaotic) are the **Cynefin** domains (Snowden), uncredited in the talk. Transcript pulled via `yt-dlp` captions — the timestamped DART definition is at ~14:33.

**Systems-thinking lens** (iceberg, stocks/flows, feedback loops, delays, emergence, information flows, leverage points) is Donella Meadows' canon — *Leverage Points: Places to Intervene in a System* (1999) and *Thinking in Systems* (2008). Verified via deep-research adversarial pass (3-vote). The "Don't overclaim" section encodes claims that pass refuted that pass 0-3 (no strict ordering of middle leverage points; goal-change is not universally the top lever; systems thinking is not a fixed N-concept schema).
