# Grill Protocol

Shared interview procedure for `grill-me` and `grill-with-docs`.

## Procedure

Interview the user relentlessly about every aspect of the plan until you reach
shared understanding. Walk down each branch of the design tree, resolving
dependencies between decisions one-by-one. For each question, provide your
recommended answer.

**Ask the questions one at a time.** Wait for feedback on each question before
continuing. Rushing ahead wastes tokens on questions whose answer the user has
already implied in a prior response.

If a question can be answered by exploring the codebase, explore the codebase
instead.

## When to stop

End the session when:
- The user says the plan is ready
- All major branches have been explored
- No new concerns surface after two rounds of "anything else?"
